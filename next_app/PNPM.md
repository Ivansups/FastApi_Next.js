# Работа с pnpm в Next.js проекте

## Основные команды

### Установка зависимостей
```bash
cd next_app
pnpm install
```

### Запуск в режиме разработки
```bash
pnpm dev
```

### Сборка для продакшена
```bash
pnpm build
```

### Запуск продакшен версии
```bash
pnpm start
```

### Линтинг кода
```bash
pnpm lint
```

## Структура:

- `pnpm-lock.yaml` - файл блокировки версий (хранится в git)
- `.next/` - папка сборки (игнорируется в git)
- `node_modules/` - зависимости (игнорируется в git)

## Полезные команды pnpm:

```bash
# Проверить устаревшие пакеты
pnpm outdated

# Обновить зависимости
pnpm update

# Удалить все зависимости и переустановить
pnpm install --force
```
